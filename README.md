# Static-rb

A very stripped-down HTTP server built with Ruby and Rack, to serve static assets.

## Prerequisite

* Ruby 2.6
* Docker (optional)
* Compose (optional)

## Installation

```bash
bundle install
```

OR with Docker

```bash
docker-compose build
```

## Usage

```bash
bundle exec rackup
```

OR with Docker

```bash
docker-compose up
```

The website is available at [http://localhost:3000](http://localhost:3000)
